# Shopping Management System

A console-based Shopping Management System developed in **C++** that enables users to manage products, maintain inventory, process customer orders, and store product information using file handling.

> **Academic Project**
>
> This project was developed during my **1st Semester** as part of the **Computer Fundamentals** course at **Bahria University**.
> It has been preserved on GitHub to document my learning journey and demonstrate my programming progress over time.

---

## Features

- Product Management
  - Add Product
  - Remove Product
  - Search Product

- Inventory Management
  - View Available Stock
  - Add Stock

- Shopping Cart
  - Add Products to Cart
  - Remove Products from Cart
  - View Cart

- Checkout System
  - Generate Receipt
  - Calculate Total Bill
  - Apply Discount

- File Handling
  - Load product information from a text file
  - Save updated inventory automatically
