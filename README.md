## 📌 About

This project was developed during my **1st Semester** at **Bahria University Islamabad** as part of the *Computer Fundamentals* course. It is preserved on GitHub to document my early learning and showcase my growth as a Computer Engineering student.

---

## ✨ Features

| Module | Functionality |
|---|---|
| 🏷️ **Product Management** | Add, remove, and search products |
| 📦 **Inventory Management** | View stock levels and restock items |
| 🛍️ **Shopping Cart** | Add/remove items, view cart summary |
| 💳 **Checkout System** | Generate receipts, calculate totals, apply discounts |
| 💾 **File Handling** | Auto-load and save data via `products.txt` |

## 🚀 Run Locally

**1. Clone the repository**
```bash
git clone https://github.com/muhammadshoaib-ce/Shopping-Management-System.git
```

**2. Open in Visual Studio 2022**
```
File → Open → Project/Solution → Shopping_Management_System.slnx
```

**3. Build & Run**
```
Ctrl + Shift + B   →   Build Solution
Ctrl + F5          →   Run without Debugging
```

> ⚠️ Make sure `products.txt` is in the same directory as the compiled `.exe`
